# 🚀 Smart Resume Builder

Create professional, ATS-friendly resumes in minutes with live preview, multiple templates, profile photo support, and one-click PDF export.

## 🌐 Live Demo

**Try it now:**
https://smart-resume-builder1.netlify.app/

---

## ✨ Features

### 👤 Personal Information

* Full Name
* Job Title
* Email
* Phone Number
* Location
* LinkedIn Profile
* GitHub Profile
* Portfolio Website
* Profile Photo Upload

### 📝 Professional Summary

* Dedicated summary section
* Real-time preview updates

### 🎓 Education

* Add multiple education entries
* School/College Name
* Degree/Class
* Start & End Year

### 💼 Experience

* Add unlimited work experiences
* Company Name
* Role/Position
* Start & End Date
* Experience Description

### 🛠 Skills

* Skills as tags
* Easy add/remove functionality
* ATS-friendly formatting

### 🚀 Projects

* Project Name
* Technology Stack
* Live Demo Link
* GitHub Repository Link
* Project Description

### 🏆 Certifications

* Certificate Name
* Issuer
* Completion Year

### 🎨 Resume Templates

* Modern
* Minimal
* Professional
* Executive
* Creative
* Tech
* Elegant
* Classic
* Sidebar
* Compact

### 📄 PDF Export

* One-click download
* A4 optimized layout
* Professional formatting

### 💾 Data Persistence

* Auto-save using localStorage
* Manual save option
* Clear all data functionality

### 📱 Responsive Design

* Desktop friendly
* Tablet optimized
* Mobile responsive
* Smooth UI animations

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/Smart-Resume-Builder.git
```

### Open Project

```bash
cd Smart-Resume-Builder
```

### Run

Simply open:

```text
index.html
```

No backend or installation required.
